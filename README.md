# Projeto Linguagens Formais e Autômatos

## Tecnologias Utilizadas

Linguagem de programação utilizada no projeto  

## Descrição

Este projeto consiste na implementação de conceitos fundamentais da área de Linguagens Formais e Autômatos, com foco na análise e validação de gramáticas, construção de autômatos e definição de expressões regulares.

A aplicação simula o comportamento de linguagens formais, permitindo verificar regras gramaticais e representar formalmente padrões de cadeias.

## Funcionalidades

Verificação de gramáticas formais  
Construção de autômatos finitos  
Representação de linguagens por meio de expressões regulares  
Validação de cadeias de entrada  

## Objetivo

O objetivo do projeto é aplicar na prática os conceitos teóricos da disciplina de Linguagens Formais e Autômatos, proporcionando uma melhor compreensão sobre reconhecimento de padrões, estruturas formais e funcionamento de autômatos.
